## 📖 Overview

The **Romantic Surprise Website** is a lightweight front-end project designed to create a memorable and interactive experience through beautiful visuals and smooth animations. It demonstrates modern web design principles while remaining simple, responsive, and easy to customize.

Whether you're learning front-end development or looking for inspiration for creative landing pages, this project serves as a great example.

---

## ✨ Features

- ❤️ Elegant romantic UI
- 🎨 Beautiful pink gradient background
- 💖 Floating heart animations
- 🎁 Interactive surprise message
- 📱 Responsive design
- ⚡ Smooth CSS transitions
- 💻 Beginner-friendly code structure
- 🌐 Works in all modern browsers
